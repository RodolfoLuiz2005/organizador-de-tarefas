# Activity Flow — Organizador de Atividades

Sistema completo feito com HTML, CSS e JavaScript puro.

## Arquivos principais

- `index.html`
- `style.css`
- `script.js`

## Funcionalidades

- Cadastro de atividades
- Duração em minutos, horas ou dias
- Edição de atividades
- Exclusão com confirmação
- Status pendente/concluída
- Filtros por status
- Resumo geral
- Tempo total planejado
- Persistência com localStorage
- Animações de entrada, cards, exclusão e rolagem
- Responsividade para desktop, tablet e celular
- Estrutura preparada para migração futura para Firebase

## Como testar

Abra o arquivo `index.html` no navegador ou use o Live Server no VS Code.
